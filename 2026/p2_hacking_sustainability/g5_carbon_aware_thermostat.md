summary: "For project 2 we want to implement a carbon aware thermostat controller for Home Assistant ([https://www.home-assistant.io](https://www.home-assistant.io)). The controller should shift the temperature based on the current energy grid (how clean it is: solar energy, wind energy or fossil fuel). The controller will use carbon intensity data, weather forecasts, and indoor temperature as inputs for its algorithm. The project will involve API integration, Home Assistant plugin development and a simple probabilistic algorithm to update the temperature. We will compare the CO2 emission of a normal virtual thermostat, with the carbon aware version."
